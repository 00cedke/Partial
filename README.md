# Partial
Wii U website for message conversation

# Execute server
- Install NodeJS with `sudo apt install nodejs` then, `sudo apt install npm`
- `git clone https://github.com/00cedke/Partial`
- `cd Partial-main`
- `node server.js` to run the web server.
